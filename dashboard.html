LinkedIn par job kaise post karein (guide)
  1. Yahan job Save karein aur status Open rakhein.
  2. Jobs table se apply link copy karein — yeh URL LinkedIn par paste hogi.
  3. LinkedIn → Jobs → Create job — title aur JD yahan se copy karein.
  4. Easy Apply band karein → Apply on company website select karein.
  5. External URL mein apply link paste karein, e.g. apply.html?job=junior-backend-developer
  6. Candidate apply karega → careers page → CV upload → AI screening (24/7).
