LinkedIn par job kaise post karein (guide)
  1. Yahan job Publish karein — status Open hona chahiye.
  2. Copy apply link dabayein — yeh URL LinkedIn par paste hogi.
  3. LinkedIn → Jobs → Create job — title aur JD portal se copy karein.
  4. Easy Apply band karein → Apply on company website select karein.
  5. External URL mein apply link paste karein, e.g. apply.html?job=junior-net-developer
  6. Candidate Apply dabayega → aapki site → CV → AI screening (24/7).
